PullSlider JQuery

a Project by Bohdan Anderson, this is a tool to create mobile slide down menus, the main case example is to allow a site go from a bar top nav menu to a drop/pull down menu with ease. PullSlider creates the class pull-slider on the associated navigator container, when enabled and when disabled removes it, leaving your html with out a notice that pullSlider exists when not at the appropriate screen size, making it great for responsive pages

The other main focus of pull slider was the ability to pull the navigation menu like the a native OS, which we have accomplished. When initializing the sliders there are configurable parameters which are:

  • display: default is false, if the menu should start down or up, accepts boolean
  • animationSlideTime: default is 500, make sure it's the same as your transition speeds of css
  • inmode: default is true, enables pullSlider, accepts boolean
